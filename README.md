# Cloud Codex

This project is aiming to solve documentation workflow issues within the development-to-tech-support sphere where there is a constant struggle for how to store and parse documentation.

The goal is to solve this issue in a new and modern way that will increase the effectiveness of development and support staff teams across the software industry.
